# Sales Data Analysis Project

## Overview
This project demonstrates an end-to-end sales data analysis workflow using Python and Power BI.

## Features
- Data Cleaning using Pandas
- Exploratory Data Analysis (EDA)
- Sales and Profit Analysis
- Data Visualization
- Interactive Power BI Dashboard

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Power BI
- Google Colab

## Project Outcome
This project provides business insights through data cleaning, analysis, and visualization to support data-driven decision-making.
